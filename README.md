# Features
👤 User registration (name, email, password)
🛍️ Browse products by category (Men, Women, Children)
🛒 Add items to cart
❌ Remove items from cart
📦 View cart items
💰 Calculate total bill
💳 Multiple payment methods:
UPI
Card
GPay
PhonePe
Cash on Delivery (COD)

# OOP Concepts Used
Encapsulation → User, Product, Cart classes
Inheritance → Admin inherits from User
Polymorphism → Different payment methods (pay() method)
Abstraction → Payment abstract base class
